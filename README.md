\# BTK Datathon 2026 — Career Success Score Prediction



This repository contains my solution workflow for BTK Datathon 2026, focused on predicting students' `career\_success\_score` values.



\## Problem



The task is a regression problem where the goal is to estimate a continuous career success score for each student profile.



\- Target: `career\_success\_score`

\- Metric: Mean Squared Error (MSE)

\- Data type: Tabular + natural language text

\- Key text feature: `mentor\_feedback\_text`



\## Dataset



The dataset includes academic, technical, project, portfolio, interview, GitHub, career preparation, and soft skill related features.



Competition data files are not included in this repository because they are subject to competition rules.
